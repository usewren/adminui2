# WREN Admin UI

The browser-based administration interface for [WREN](https://wren.aemwip.com).

Vanilla JavaScript — no build step, no framework dependencies. Served at `/admin` on any WREN instance.

## Features

- Browse and manage collections, documents, and tree structures
- Version history, diffs, and rollback
- JSON Schema editor and validation
- Binary asset upload and preview
- API key management
- Collaborator invites and org switching
- Permission rule editor
